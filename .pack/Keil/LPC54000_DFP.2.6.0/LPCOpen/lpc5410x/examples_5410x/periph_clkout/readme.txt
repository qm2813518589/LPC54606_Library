LPC5410x Clock output example
========================================================

Example description
-------------------
This example shows how to select a clock output and divider to
route to the CLKOUT signal. It prints the name of the current selected
clock and by pressing a key in UART terminal the selection can be
changed. UART must be configured for 115200,8n1 configuration.

Special connection requirements
-------------------------------
Connect an oscilloscope to pin 17 of digital connector J2.
